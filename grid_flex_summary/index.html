Flexbox

With flex grow you can manage the remaining amount of space.

    .flexsize {
      display: flex;
      border: 2px solid darkmagenta;
    }

    .flexsize a {
      background-color: aliceblue;
      border-right: 2px solid grey;
      min-width: 10vw;
    }

    .flexsize a:first-child {
      flex-grow: 1;
    }

Justify-content justifies the whole flexbox, align-items adjusts the flexitems vertically

    .justalign {
      display: flex;
      flex-wrap: wrap;
      border: 2px solid darkmagenta;
      justify-content: space-evenly;
      align-items: flex-end;
      min-height: 400px;
    }

    .justalign .image1 {
      height: 150px;
    }

    .justalign .image2 {
      height: 100px;
    }

    .justalign img {
      height: 300px;
      object-fit: cover;
    }

Flex-shrink shinks a flex element when it is smaller than the element's flex-basis.

    .growshrink {
      display: flex;
      border: 2px solid darkmagenta;
      height: 200px;
    }

    .growshrink div {
      flex-basis: 300px;
      display: flex;
      align-items: center;
      justify-content: center;
      font-size: 3rem;
    }

    .growshrink .box1 {
      background-color: navy;
      color: #fff;
    }

    .growshrink .box2 {
      background-color: lightcoral;
      flex-grow: 1;
      flex-shrink: 6;
    }

Calc is handy when you add borders to an element in flex boxes. Let's say your elements add up perfectly without borders. But when you add borders your design breaks because your elements are too wide. You can then substract the borders from the elements width.

    .calc {
      border: 2px solid darkmagenta;
      display: flex;
      height: 200px;
    }

    .calc div {
      background-color: hotpink;
      min-width: calc(33.333% - 40px);
      text-align: center;
      font-size: 3rem;
      display:flex;
      justify-content: center;
      align-items: center;
      color: #fff;
      border: 20px solid #69b4ff;
    }

CSS Grid

Auto works similar to flex-grow. It allocates the remaining space to the other grid elements relative to the element's size.

    .auto {
      border: 2px solid darkmagenta;
      display: grid;
      grid-template-columns: auto 1fr auto;
      grid-template-rows: 50px 100px;
      grid-gap: 20px;
    }

    .auto div {
      background-color: chocolate;
    }

With grid-column and grid-row you can playe elements on specific positions on your grid. Span allows you to extend your grid items over multiple columns or rows. With grid-auto-flow: dense you can fill up the empty grid items so that your grid does not have empty holes.

    .specific {
      display: grid;
      grid-gap: 10px;
      grid-template-columns: repeat(2, 1fr 2fr 5fr);
      grid-template-rows: repeat(1, 50px 100px 20px);
      grid-auto-flow: dense;
      grid-auto-rows: 300px;
      border: 2px solid darkmagenta;
    }

    .specific div {
      background-color: violet;
      display: flex;
      justify-content: center;
      align-items: center;
      color: #fff;
    }

    .specific .box13 {
      grid-column: 1 / span 4;
      background-color: blue;
    }

    .specific .box17 {
      grid-row: span 2;
      background-color: palegreen;
    }

    .specific .box18 {
      grid-column: 1 / span 3;
      background-color: crimson;
    }
